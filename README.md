## Who I Am

Senior Software Engineer with 6+ years building systems that can't afford to go down. Banking platforms, high-traffic mobile apps, and middleware that moves real money and real data at scale. I have a Master's in Data Science (UCJC) and a Telematics Engineering degree (PUCMM), and that combination has pushed most of my work and thinking toward the intersection of backend engineering and AI.

Beyond shipping production software, I do sometimes teach at PUCMM, speak at conferences: JConfDominicana, DevNexus, Azul Systems AI4J, and Java User Groups across the globe — and help organize JConfDominicana every year. I co-lead the Dominican Republic Java User Group, and have been featured in Open Source Global Voices as a Caribbean technology contributor. When time allows, I contribute to open source.

---

## What I've Built

Most of my experience comes from working on systems with real operational weight. A few examples:

- **Banking core systems.** Backend work on high-volume platforms processing millions of transactions weekly. The job is keeping them correct and stable, not just functional.
- **Mobile app for the DR's largest telecom.** Backend engineering for an app with around 1 million users. Performance issues here are not abstract.
- **Email middleware at scale.** Led the architecture and full implementation of a cloud-native system on Java and AWS that handles 250M+ emails per year for a global healthcare data company.
- **Spring Boot 2 to 3 migration.** Led this on a live production platform. The goal was security and performance. The constraint was keeping everything running while doing it.
- **Enterprise SaaS integrations.** ERP, CRM, and third-party integrations across multiple industries using enterprise integration patterns, microservices, and event-driven designs.

Working on these has made me good at debugging things that aren't supposed to break. Tracing slow queries, tracking down race conditions, reading production logs at odd hours. That's where a lot of the real learning happens.
